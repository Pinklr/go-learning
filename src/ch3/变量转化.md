## 变量转化

**不支持隐式类型转化**  别名的隐式类型转换也不被允许

```go
package type_test

import "testing"

type Myint int32

func TestType(t *testing.T) {
	var a int32 = 1
	var b int64 = 2
	a = b
	var c Myint = a

}

```

	t.Log(a, aPtr)
